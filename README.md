# InAir4 Hookup Guide
The Modtronix InAir4 module 

# Pinouts
|--------|:-----:|--------:|
| GND	 |  		|	NC 		|
| CS	 |   		|	3.3V	|
| DIO3	 |		|	SCK		|
| RST	 |		|	MISO	|
| DIO0 	 |		|	MOSI	|
| DIO1 	 |		|	NC		|
| DIO2 	 |		|	VSWTICH	|


# Pinout Description

|--------|--------:|
| GND	 | Ground PIN |  
| CS	 | Chip Select |  	
| DIO3	 | Digital Input / Output (3) |		
| RST	 |	Reset Pin |	
| DIO0 	 | Digital Input / Output (0) |		
| DIO1 	 | Digital Input / Output (1) |		
| DIO2 	 | Digital Input / Output (2)		
|--------|:-----:|--------|
| NC	 | Not Connected | 	
| 3.3V	 | Vin, 3.3VDC  |	
| SCK	 |	SPI Clock	|
| MISO	 |	SPI MISO	|
| MOSI 	 | SPI MOSI		|
| NC 	 |	Not Connected |	
| VSWITCH 	 |		VSWITCH |
